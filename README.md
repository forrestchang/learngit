## 实战

1. fork 本仓库
2. clone fork 后的仓库到本地
3. 建立文件夹，格式「姓名拼音」，例如：lijunping
4. 从 [problems.md](./problems.md) 中挑选一个问题写代码，放到建立的文件夹中
5. 推送修改到 fork 后的仓库，并向原始仓库提交 pull request
